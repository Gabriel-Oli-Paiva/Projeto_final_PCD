# Enigma, no Python!?

## Projeto final de Prática em Ciência de Dados

**Aluno**: Gabriel de Oliveira Paiva

**Professores**: Dr.Daniel R. Cassar, James Moraes de Almeida e Leandro Nascimento Lemos

**Instituição**: Ilum Escola de Ciência.

**Introdução**: A máquina Enigma foi um dispositivo de criptografia utilizado na segunda guerra mundial, ela funcionava de forma parecida com uma máquina de escrever porém quando a pessoa apertava uma tecla a letra era substituída por outra. A troca era feita em um padrão que dependia de algumas configurações iniciais, ao digitar um texto criptografado em outra Enigma com as mesmas configurações a mensagem era descriptografada e revelada.

**Resumo**: Este projeto consiste na ideia de utilizar os conhecimentos fornecidos em sala de aula para o desenvolvimento de dois modelos em Python da máquina Enigma, sendo um com a ideia de se aproximar ao máximo do funcionamento que a máquina apresentava e outro com o intuito de usar a Enigma como base e ampliar seu funcionamento.

## Pastas Importantes: 
  
  **Projeto_de_PCD**: é a pasta que contém todas as explicações sobre o projeto e os códigos para a execução das duas Enigmas
  
  **Enigma modificada**: é a pasta que contém apenas as funções e a própria Enigima modificada
  
  **Enigma verossímil**: é a pasta que contém apenas as funções e a própria Enigima verossímil

## Tecnologias usadas:

Python na versão 3.13.7
